# Assets Directory

This directory contains static assets for the portfolio website.

## Required Files

### Social Sharing Image (og-image.jpg)

Create an Open Graph image for social media sharing:
- **Recommended size:** 1200x630 pixels
- **Format:** JPG or PNG
- **File name:** `og-image.jpg`
- **Content suggestions:**
  - Your name or brand
  - Your title/role
  - Key technologies or skills
  - Professional design that represents your personal brand

You can create this using tools like:
- [Canva](https://www.canva.com/) - Free design tool
- [Figma](https://www.figma.com/) - Professional design tool
- Adobe Photoshop or similar

Once created, place the file here: `assets/og-image.jpg`

The image is referenced in the HTML meta tags for social sharing on platforms like:
- LinkedIn
- Twitter/X
- Facebook
- Discord
- Slack

## Project Images

Add screenshots of your projects here:
- Use descriptive filenames (e.g., `project-ai-npc-system.jpg`)
- Optimize images for web (use tools like TinyPNG)
- Recommended dimensions: 600x400px for project cards
- Keep file sizes under 500KB for fast loading

## Other Assets

You can also add:

- Apple touch icons for iOS devices
- Project demo videos or GIFs
- Resume PDF
